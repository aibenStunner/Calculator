# Scientific-Calculator
A simple scientific calculator.

## Motivation :seedling:

## Installation :package:
1. Clone the repo
```bash
   git clone https://github.com/aibenStunner/Calculator.git
   cd Calculator
```

## Usage :computer:

